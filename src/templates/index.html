The API reference of the backend

  • /stocks : Return all stocks
  • /stocks/last : Return the last stock of each company
    • ?comp=[Tag of the company] : eg: stocks only of Telef√≥nica if Tag is TEF
    • ?from=[datetime in isoformat] : eg: stocks after "%Y-%m-%d %H:%M:%S" eg:"2005-09-01 12:30:09"
  • /papers : Return all papers
  • /papers/[num] : Return the [num] first articles from all the websites. num = 1..99
  • /head_lines : Return all papers head_lines
  • head_lines/[num] : Return head_lines of the [num] first articles from all the websites. num = 1..99
    • ?from=[datetime in isoformat] : articles after "%Y-%m-%d %H:%M:%S" eg:"2005-09-01 12:30:09"
    • ?web_site=[url] : only articles from the selected url eg:"http://www.elmundo.es"
